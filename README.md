# Customer Query Intent Classification using NLP

## 📌 Project Overview

This project focuses on automatically classifying customer support queries into predefined intent categories using Natural Language Processing (NLP) and Machine Learning techniques.

The system helps businesses understand customer concerns more efficiently by identifying the intent behind customer messages such as refund requests, order tracking, payment issues, complaints, cancellation requests, technical support, and more.

The project was implemented using Python, TF-IDF Vectorization, and Logistic Regression, along with multiple professional visualizations for analysis and interpretation.

---

# 🎯 Objectives

- Automate customer query classification
- Improve customer support efficiency
- Generate customer and market insights
- Analyze customer behavior patterns
- Visualize intent distributions and model performance

---

# 🧠 Intent Categories

The model classifies customer queries into the following intents:

- order_status
- refund_request
- cancel_order
- payment_issue
- complaint
- return_request
- product_info
- account_issue
- technical_support
